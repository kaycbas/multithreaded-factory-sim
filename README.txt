This is a factory simulation written in Java. The purpose of this project is practice multithreading, GUI's, file parsing, and software design principles.

To run this program, open the project in Eclipse, build it, and run it. Once the factory GUI displays, hit the 'Open File' option and select the included
'Factory' folder. This will supply the values for number of workers, number of tools, number of resources, etc. to run the simulation with.
